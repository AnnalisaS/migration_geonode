How to translate the documentation
==================================

1. As a firt step, generate all of the needed .pot files::

    cd docs
    sphinx-build -b gettext . i18n/pot
  
2. Run the pre_translate.sh script to generate all of the .po files for each language::

    $ sh scripts/pre_translate.sh
  
3. Now edit the .po files you need, make the translations and then run the post_translate.sh script. There are two workflows, the first using your favourite editor, the second using the transifex ui. Using your own editor, first pull from transifex, then change the files, pull files to transifex and then compile the messages using the post_translate.sh script::

    $ tx pull -a
    $ vi i18n/it/LC_MESSAGES/index.po
    $ tx push -s -t
    $ sh scripts/post_translate.sh
  
4. Finally build your html or pdf, and you should get the localized versions as well::

    $ make html LANG=it
    
5. In case master (English) doc is updated, regenerate the .pot files (step 1), the corresponding .po files will be merged, and then continue with the same workflow
